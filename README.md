# OpenClaw Channel Automation (Public)

Public project with reproducible setup for Telegram channel operations via OpenClaw + Telegra.ph.

## What is included
- Detailed articles (architecture + fine-tuning)
- Playbooks for safe publication flow
- Public-safe scripts for Telegram transport checks/publishing
- Custom local skills for copywriting and Telegra.ph publishing
- Channel archive samples with article/post pairs

## Safety
This repository must not contain secrets.
Use environment variables for all credentials/tokens.

## Structure
- `docs/articles/` — published article sources
- `docs/playbooks/` — operational runbooks
- `docs/plans/` — roadmap and architecture notes
- `scripts/` — operational helper scripts
- `skills/` — custom AgentSkills used in this setup
- `channel-archive/` — example per-publication folders
